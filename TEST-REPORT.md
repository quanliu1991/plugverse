## 🐛 已修复的 Bug

### Bug 1: Path 未导入
**问题:** `app/main.py` 中使用了 `Path` 但未导入  
**修复:** 添加 `from pathlib import Path`  
**影响:** 应用无法启动

### Bug 2: 全局变量未初始化
**问题:** `plugin_manager` 在 lifespan 外使用导致 NameError  
**修复:** 修改 `load_installed_plugins()` 和 `unload_all_plugins()` 函数，将 `plugin_manager` 作为参数传递  
**影响:** 插件无法加载

### Bug 3: API 路由缺失
**问题:** `/api/health`, `/api/tasks`, `/api/files` 等路由未定义  
**修复:** 添加完整的 RESTful API 路由  
**影响:** 前端无法获取数据

### Bug 4: 返回格式不一致
**问题:** `list_plugins()` 返回数组而非对象  
**修复:** 统一返回格式为 `{"plugins": [...]}`  
**影响:** 前端解析错误

### Bug 5: vue-tsc 构建失败
**问题:** vue-tsc 与当前 TypeScript 版本不兼容  
**修复:** 将类型检查移至独立脚本 `build:types`  
**影响:** 前端无法构建
